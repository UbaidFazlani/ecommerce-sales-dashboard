E-Commerce Sales Performance Dashboard

This repository contains an interactive Excel dashboard designed to analyze e-commerce sales performance using transactional sales data.

The dataset used in this project was sourced from Kaggle and includes information such as order details, sales amount, quantity, product category, platform, brand, city, customer ratings, and order dates.

Dashboard highlights

Key performance indicators (Total Sales, Total Orders, Quantity Sold, Average Order Value, Average Rating)

Monthly sales trend analysis

Sales distribution by category, platform, city, brand, and product

Identification of top-performing products and brands

Interactive slicers for dynamic filtering and analysis

This dashboard provides a consolidated view of sales performance and supports data-driven insights across multiple business dimensions.

Project Objective

The objective of this project is to analyze e-commerce sales data and develop an interactive dashboard that delivers clear insights into overall sales performance, trends over time, 
and key contributing factors such as category, platform, brand, product, and city. The dashboard is designed to support data-driven decision-making by presenting key metrics and 
performance insights in a clear and accessible format.
